# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| &lt; 1.0   | :x:                |

## Reporting a Vulnerability

Use `issues` page to report a vulnerability. If you found a vulnerability, create an issue at [here](https://github.com/aiocat/request/issues/new)
